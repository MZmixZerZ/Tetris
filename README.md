# Tetris — Java

เกม Tetris ที่เขียนด้วย Java ไม่มี library นอก ใช้แค่ JFrame กับ Swing

## วิธีรัน

ต้องมี JDK ติดตั้งอยู่ก่อน จากนั้น

```bash
javac Tetris.java
java Tetris
```

## วิธีเล่น

| ปุ่ม | ทำอะไร |
|------|--------|
| `←` `→` | เลื่อนซ้าย / ขวา |
| `↑` | หมุนชิ้น |
| `↓` | เลื่อนลงเร็ว |
| `Space` | ดรอปลงทันที |
| `R` | เริ่มเกมใหม่ (หลัง Game Over) |
